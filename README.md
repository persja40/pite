Python in the Enterprise - car sharing
=========================================

# How to run locally
- install all requirements
```bash
pip install -r /path/to/requirements.txt
```
- run app locally from project folder
```bash
python3 manage.py runserver
```
- it's accessible at http://127.0.0.1:8000/
